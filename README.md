# rfc_jwt

RFC Research Related to JSON Web Tokens (JWT)

## Use cases

* Authentication
* Information exchange

## JWT structure

* JOSE Header
* JWT Claims
  * "exp" (Expiration time) Claim
* Signature / Authentication Tag
